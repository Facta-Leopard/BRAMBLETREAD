# BrambleTread

젊은 시절의 브람블 중사가 마정석 기동 소형 요새 `Treadmill`을 지휘해 참호 전선을 돌파하는 iPhone 세로 모드 SpriteKit 프로토타입이다.

## 문서

- [개인정보 처리방침](Docs/PrivacyPolicy.md): App Store Connect 개인정보 처리방침 URL에 연결할 공개 문서.
- [지원 문서](Docs/Support.md): App Store Connect 지원 URL에 연결할 공개 문서.
- [App Store Connect URL 정리](Docs/AppStoreConnectURLs.md): 개인정보 처리방침과 지원 URL 입력값.

## App Store Connect 공개 URL

- 개인정보 처리방침: https://facta-leopard.github.io/BRAMBLETREAD/Docs/PrivacyPolicy.html
- 지원 URL: https://facta-leopard.github.io/BRAMBLETREAD/Docs/Support.html

개인정보 처리방침과 지원 문서는 언어별 파일을 분리하지 않고, 한 페이지 안에서 한국어와 영어를 함께 제공한다.

## 현재 프로토타입 범위

- iPhone 17 기준 세로 모드 SwiftUI + SpriteKit 구조.
- 스플래시, 첫 실행 온보딩, 정비창 메인, 전차 내부, 인게이지, PX, 통신암호실, 모스 가젯 타전 뷰.
- PX 보상형 광고, Game Center 점수 제출, 한국어/영어 로컬라이즈 구조가 준비되어 있다.
